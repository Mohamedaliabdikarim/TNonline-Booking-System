# TNonline Booking System

A complete cross-platform booking and analytics system developed during my role as System Developer at **Capplon AB**. Built from scratch with .NET MAUI and Blazor Hybrid for Windows, Android, and iOS.

> **Note:** Source code is proprietary to Capplon AB and cannot be shared publicly. This repository documents the project, my role, and the technical work I delivered.

---

## 🎯 What it does

- Customer login and role-based access control
- Booking management and scheduling
- Real-time data visualization and dashboards
- PDF report generation for business analytics
- SQL-driven backend for statistics and reporting

## 🛠️ Tech Stack

| Area | Technology |
|------|-----------|
| Framework | .NET MAUI + Blazor Hybrid |
| Language | C# |
| Database | SQL Server + ADO.NET |
| Reporting | QuestPDF (PDF), custom SVG-based charts |
| Architecture | Dependency Injection, service layer pattern |
| DevOps | Docker + Azure, CI/CD pipeline |
| Integrations | REST APIs, webhook-based data flows |

## 🏗️ My contributions

- **Architecture:** Designed a scalable service layer with Dependency Injection — improved modularity, testability, and long-term maintainability
- **Data layer:** Built robust data flows with SQL Server and ADO.NET (parameterized queries, transaction handling, advanced data generation for analytics)
- **Reporting:** Developed professional reporting solutions with QuestPDF and custom SVG-based dashboard components
- **Security:** Implemented authentication, role management, and stable state management
- **Integrations:** Developed API and webhook-based data flows for syncing products, orders, and customer data
- **Quality:** Defined acceptance criteria and integrated quality gates into the CI pipeline
- **Deployment:** Containerized the application with Docker and deployed to Azure
